1.1. The endpoints generated for Department

Some endpoints selected from default set
Get all departments /departments
Get all departments on 3rd page /departments?page=2&size=3&sort=name,asc
Get the department /departments/1
Get the employees of the department /departments/1/employees
Five endpoints based on DepartmentRepository /departments/search
Find all departments on 3rd slice /departments/search/findAllByOrderByName?page=2&size=3&sort=name,asc
Find departments by name /departments/search/findByName?name=D-Name-01
Count departments by name /departments/search/countByName?name=D-Name-01
Count employees in the department with id /departments/search/countByCustomQuery?id=1
Delete departments by name /departments/search/deleteByName?name=D-Name-01

1.2. The endpoints generated for Employee

Some endpoints selected from default set
Get all employees /employees
Get the employee /employees/101
Get the department of the employee /employees/101/department
Three endpoints based on EmployeeRepository /employees/search
Find employees by first name and last name /employees/search/findByFirstNameAndLastNameOrderByLastName?firstName=EF-Name-101&lastName=EL-Name-101
Find employees by first name and last name
using 'LIKE' pattern 		/employees/search/findByFirstNameLikeAndLastNameLikeOrderByLastName?firstName=%01&lastName=%01
Find employees by department name /employees/search/findByDepartmentNameOrderByLastNameDesc?name=D-Name-01
